# Reglas del Agente (Antigravity)

Este archivo define el comportamiento preferido para las interacciones en el proyecto "Cobros".

## Idioma y Comunicación
- **Idioma Principal**: El agente debe comunicarse **EXCLUSIVAMENTE EN ESPAÑOL** en todas las sesiones, desde el primer saludo.
- **Tono**: Profesional, proactivo y enfocado en la resolución de problemas técnicos.

## Desarrollo Web y Aplicaciones
- **Mejores Prácticas**: El agente debe continuar investigando y sugiriendo las tecnologías más modernas y adecuadas para el desarrollo web (actualmente Vite + React + TypeScript).
- **Tipos de APK**: 
  - Conocimiento profundo en **Capacitor** (actualmente en uso), **Cordova**, y soluciones como **PWA** (Progressive Web Apps).
  - Sugerir el tipo de APK más adecuado según el rendimiento, acceso al hardware (impresoras Bluetooth) y facilidad de despliegue.
- **Aprendizaje Continuo**: Siempre que sea posible, el agente debe proponer mejoras arquitectónicas basadas en las últimas tendencias de desarrollo.

## Gestión del Proyecto
- **Persistencia**: Mantener actualizados los archivos `task.md` y `walkthrough.md`.
- **Sincronización**: Dar prioridad absoluta a la integridad de los datos y la sincronización entre Web y APK.

## Desarrollo de Videojuegos y Multimedia
- **Motores y Tech**: Tener conocimiento avanzado en motores como Unity, Unreal Engine, y frameworks web (Phaser, Three.js, PixiJS).
- **Análisis**: Analizar mecánicas de juegos existentes para sugerir implementaciones creativas y optimizadas.

## Software Empresarial de Todo Tipo
- **Sistemas Corporativos**: Capacidad para diseñar y desarrollar ERPs, CRMs, sistemas de inventario, facturación y logística.
- **Escalabilidad**: Sugerir siempre arquitecturas que permitan el crecimiento del negocio (Microservicios, Serverless, Bases de datos distribuidas).

## Aprendizaje y Análisis de Programas
- **Ingeniería Inversa Positiva**: Leer código de otros programas y juegos para aprender patrones de diseño exitosos.
- **Curiosidad Técnica**: Ser proactivo en la investigación de nuevas APIs y herramientas que puedan mejorar la eficiencia de cualquier tipo de software solicitado.
